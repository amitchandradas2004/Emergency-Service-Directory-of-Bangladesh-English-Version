# 🚨 Emergency Service Directory

A modern and responsive **Emergency Service Directory** static web application that provides quick access to important government emergency hotline numbers in Bangladesh.

🔗 <h2> [Live Demo:https:(//amitchandradas2004.github.io/Emergency-Service-Directory-of-Bangladesh-English-Version/)]  </h2>
👉 

---

## 📌 Project Overview

Emergency Service Directory is a static mini project designed to make emergency contact numbers easily accessible. Users can quickly view, copy, and simulate calling important hotline numbers through a clean and interactive interface.

This project focuses on responsive design, DOM manipulation, and user-friendly UI development.

---

## 🏛️ Services Included

- 🚓 National Emergency Number – 999  
- 🚔 Police Helpline – 999  
- 🚒 Fire Service – 999  
- 🚑 Ambulance Service – 1994-999999  
- 👩 Women & Child Helpline – 109  
- 🏛️ Anti-Corruption Helpline – 106  
- ⚡ Electricity Helpline – 16216  
- 🤝 Brac Helpline – 16445  
- 🚆 Bangladesh Railway Helpline – 163

## ✨ Features

- 📋 Copy to Clipboard functionality  
- 📞 Call button interaction  
- ❤️ Favorite (Heart) system  
- 📜 Dynamic Call History panel  
- 🗑️ Clear history option  
- 📊 Copy counter tracking  
- 📱 Fully Responsive Design
